# IMG.LY CE.SDK Angular Editor

Angular app with IMG.LY CreativeEditor SDK. Opens an image in the editor on load.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add your CE.SDK license key in `src/app/app.component.ts` (line 22):
```ts
license: 'your-license-key',
```

3. Run dev server:
```bash
npm start
```

Open http://localhost:4200/
